Video-based animal behaviour recognition
Published Wed 06 September 2023
With the increasing demand of animal products has brought animal welfare analysis, as more researcher involves, especially into the analysis of the social interactions inside commercial farms, For instance, disruptive inter-animal interaction, e.g. tail-biting, stepping others, etc.. These behaviors will not only affect animal welfare, as consequences, it reduces the production of the farms. Hence, the detection and prevention of such behavior became more serious. Historically, commercial farms have been using RFID sensor for monitoring, however, such technique are expensive. Therefore we explore another routs, the video-based animal behavior recognition

Recently, researchers have discovered that video surveillance is a better solution as it provides a low-cost and sustainable solution. This is also evidenced by the increasing advancements in artificial intelligence, such as Multiple Object Tracking (MOT), Joint Detection and Embedding (JDE), and Look Once (YOLO). These techniques are useful and proven to be effective. And the latest technique is MSQNet. Which eliminated the needs actor specific data, which unlocks unlimited possibility. But that’s not enough. The current difficulty is to further improve the model to the next level. Towards multi-class classification in video-base recognition.

Therefore, we propose a genius strategy. Let’s introduce addition information to the model from ground up from the available dataset (Animal Kingdom dataset). Add an addition class to it. And see if it improves the model precision and different evaluation matrices.And there’s it. By adding addition animal class (881 types) to the dataset. The further guide the model better classify the action plus the bonus animal class classification which make it classify two different things.But that’s not going to work well. Hence, we introduce another method. Let’s call it multi-phase transfer learning. By starting with just the action annotation with TimesformerInit for training, then join the animal class to it. Inspire from human learning stages.

The research shows that by introduce an addition class to the model. In our case, the animal class. Also, with multi-phase transfer learning. The precision of the action recognition does not get improve versus default model ~74%. And the animal class classification precision is better than action recognition.

With sole join two class to the model. The situation is different. The model saturated at ~72% which means the multi-phase transfer approach kept the original precision while adding addition class for the model to classified. This model now can classified animal behavior while classified the type of the animal.

With multi-phase transfer learning. The model is able to retain previous learned patterns. And without it. The model performance degraded.
